# Webbteknik 3 - Assignments (1ME323)

Detta repository innehåller kursens inlämningsuppgifter för Webbteknik 3 (1ME323).

## Struktur
- `index.html` är en landningssida som listar alla uppgifter.
- Global navigation och “kort” skapas dynamiskt med JavaScript från `assignments.js`.
- Varje uppgift ligger i en egen mapp (t.ex. `assignment1/`).

## Publicering
Projektet publiceras via GitHub Pages: https://mohamadhanoun.github.io/Webbteknik3-Uppgifter/

## Innehåll
- Uppgift 1: Samlingssida för inlämningsuppgifter
- Uppgift 2: Produktsida med kundvagn
